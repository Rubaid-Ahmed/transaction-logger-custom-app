# Instruction Manual

## Setup

1. Open Settings.
2. Set the app name, user name, and currency symbol.
3. Add the accounts the user wants to track.
4. Add or rename credit categories.
5. Add or rename debit categories.
6. Mark one credit category as the tuition category if student payments will be tracked.
7. Add student names from Students or Settings.

## Logging Transactions

1. Open Log.
2. Choose Credit or Debit.
3. Choose a category.
4. Choose an account.
5. Enter amount, date, and note.
6. Save the entry.

## Transfers

1. Open Log.
2. Use Transfer between accounts.
3. Choose the source account and destination account.
4. Enter amount, date, and note.
5. Save the transfer.

## Tuition Tracker

1. Open Tuition.
2. Choose a month.
3. Select the students to show for that month.
4. Click a day cell to mark or unmark attendance.

## Students

1. Open Students.
2. Add each student name.
3. Add cycle days only when the user wants cycle tracking.
4. Use Edit to rename or update a student.

## Backups

Use Export JSON before major changes. Use Import JSON to restore a saved dataset in the same browser or another browser.

## Hosting Note

On the hosted static version, each browser keeps its own data. For shared data on one machine or network, run the Node server version from the local app folder.
